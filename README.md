# trading-test

## To execute project run:

### create virtual env and install requirements
<pre><code>
python3 -m venv env
</code></pre>
<pre><code>
source env/bin/activate
</code></pre>
<pre><code>
pip install -r requirements.txt
</code></pre>

### run project
<pre><code>
python manage.py runserver 0.0.0.0:8000
</code></pre>
